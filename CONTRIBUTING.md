# Contributing

Thank you for contributing to `webex-ai-agents`.

## Contribution Expectations

This repository is documentation-only. Contributions should improve clarity, organization, and usefulness of the sample materials rather than introduce tooling, application code, or unrelated assets.

## Repository Rules

- Keep the repository centered on Webex AI agent and Webex Contact Center sample documentation
- Use clear folder names and README-driven navigation
- Prefer concise, professional writing over long narrative sections
- Keep sample scenarios aligned across related folders so the AI agent and flow describe the same use case

## Adding New Sample Content

When adding another sample, use the same documentation pattern:

- Create a top-level sample folder with its own `README.md`
- Add focused subfolders for the sample AI agent and the related flow
- Make sure each README explains purpose, scope, and how it connects to the rest of the example

## Pull Request Guidance

Before submitting a pull request:

- Confirm links between README files resolve correctly
- Confirm badge rows are present and visually consistent
- Confirm the documented folder structure matches the committed files
- Confirm scenario language is consistent across the root README and sample folders

## Review Standard

Changes should leave the repository cleaner, clearer, and more professional than before. Documentation that introduces ambiguity, duplicates content unnecessarily, or breaks structural consistency should be revised before merge.
